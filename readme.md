# Projeto com React

## [Vite](https://vitejs.dev/guide/)
Uma forma de criar um projeto em react de um jeito mais limpo, mais leve e mais eficiente.
```
  npm create vite@latest [nomeDoProjeto]
```
| JavaScript| TypeScript | 
| :---------| :--------- | 
| `vanilla` | `vanilla`  |
| `vue`     | `vue-ts`   |
| `react`   | `react-ts` |
| `preact`  | `preact-ts`|
| `lit`     | `lit-ts`   |
| `svelte`  | `svelte-ts`|

### Passos seguintes
Primeiro:
```
  cd [nomeDoProjeto]
```
Segundo:
```
  npm install
```
E por ultimo para rodar:
```
  npm run dev
```
